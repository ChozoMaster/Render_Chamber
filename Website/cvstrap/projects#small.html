PROJECT ONE

General Information:

Serious Games was a project course at TU Darmstadt. In groups of 2-4 students we had to work on different projects. We chose to create a visual analysis tool for mentoring collaborative multiplayer games. Therefore we created a log file format which our software is able to process and then display with the ability to interact with the data. The use case was a game called "Escape from Wilson Island" where 4 players had to work together to survive and build a boat to leave the island.
