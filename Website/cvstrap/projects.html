Bachelor Thesis - Visual Analysis of Local Correspondence in Segmentation

About:

The Bachelor-Thesis presents a new interactive system for visual and exploratory analysis of local correspondence in segmentation quality. Segmentations of several samples of one organ are analyzed on the basis of pairwise distances between a reference- and a test- mesh, which is extracted from the organ segmentation. The tool features several views on the data (Coloring, threshold based highlighting, average mesh visualization) and a set of analysis methods (clustering, cluster quality evaluation, dimension reduction) to extract new information, such as reoccurring regions or patterns of low quality segmentation. Segmentation algorithm developer can use the visual information for gaining knowledge on how their algorithms work. This insight can be beneficial to the improvement of the algorithms. The software is optimized for analyzing medical image segmentation, but can also be translated to countless domains, as it simply operates on the extracted mesh data. (Download)

Master Thesis - Interactive Dynamic Ambient Occlusion Using Approximate Voxel Cone Tracing

About:

Ambient occlusion (AO) is a relatively fast way to obtain small scale shadows which help to visually indicate curvature and spatial relationships. The industry standard to calculate this visual phenomenon is screen-space ambient occlusion (SSAO). Although many approaches in screen-space exist, the quality is still inferior to non-local techniques due to systematic errors.
This work revisited approximate voxel cone tracing, which is a non-local approach introduced by Crassin et al. in the year 2011. It supports real-time frame rates in semi dynamic scenes, which means, that geometric objects in the scene are previously classified as static or dynamic. The example scenes used by Crassin et al. usually feature a much larger amount of static than dynamic geometry. The goal of this work was to test and adapt the method for ambient occlusion in full dynamic scenes, without distinguishing between static or dynamic geometry. (Download)

Physically-based Animation & Multi-Body-Systems Tracing

About:

Physically based animation was a course at TU Darmstadt, which was accompanied by several practical projects. In groups of 2-4 students we had to develop our own simulation engine in C++. The subject of study were particle and particle systems, which are the basis of modern smoke or water simulation. Furthermore rigid bodies and multi-body-systems were discussed. With the theoretical background we programmed several simulations, which covered:

  • Simulation of a solar system
  • Hair simulation
  • Deformable soft bodies
  • Collision detection
  • Cloth Simulation
  • Mass-spring-systems

Serious Games

About:

Serious Games was a project course at TU Darmstadt. In groups of 2-4 students we had to work on different projects. We chose to create a visual analysis tool for mentoring collaborative multiplayer games. Therefore we created a generic log file format which our software is able to process and then display with the ability to interact with the data. The use case was a game called "Escape from Wilson Island" where 4 players had to work together to survive and build a boat to leave the island.

Computer Vision

About:

Computer Vision I & II are courses at TU Darmstadt with programming exercises for groups of two students. We had to implement classical and modern image processing and computer vision algorithms with Matlab. Our tasks included:

  • Perform image sharpening with Gaussian and Laplacian image pyramids
  • Detect points of interest in an image by implementing a Harris feature detector
  • Classify images with a bag of visual words approach
  • Realize image denoising and in-painting with Markov random fields
  • Track a moving object in a video using Kalman filter
